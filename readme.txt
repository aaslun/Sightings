Sightings is an easy to use plug-in that leta you geo-tag any post or page in WordPress and display them on Google maps in your blog.

== Description ==

Sightings is an easy to use plug-in that utilizes the Google Maps v3 JavaScript API to let you geo-tag any post or page in WordPress. You can display all your recorded sightings on a larger map by using the shortcode: [sightings-map] on any post or page.

Requirements:

* PHP 5.3 or later

Features:

* Easy to use
* Geo-tag any post or page
* Display geo-tag on a map on a post
* Display all recorded geo-tags collectively on a larger map by using the shortcode: [sightings-map]

== Installation ==

Manual install: Unzip the file into your WordPress plugin-directory. Activate the plugin in wp-admin.

Install through WordPress admin: Go to Plugins > Add New. Search for "Sightings". Locate the plugin in the search results. Click "Install now". Click "Activate".

== Frequently Asked Questions ==

= There is an error when i activate the plugin! What should I do? =

In your php.ini, make sure that short_open_tag is activated. If you're still experiencing trouble please contact me.
